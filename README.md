# install

> npm install

# run

* > cp config.default.js config.js (and update)
* > npm start

# use

got to:
* `http://localhost:3000/fetch` to loop fetch articles
* `http://localhost:3000/write` to write db to file
* `http://localhost:3000/authors` to display list of authors
